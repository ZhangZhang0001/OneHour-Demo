# 健身房内部管理系统 - 设计指南

## 1. 品牌定位

- **应用定位**：健身房内部管理工具
- **目标用户**：健身房员工、教练、管理人员
- **核心功能**：培训资料管理、器械巡检记录
- **设计风格**：简洁专业、功能导向

## 2. 配色方案

- **主色调**：深蓝色 (#1e40af) - 专业可信
- **辅助色**：蓝色 (#3b82f6) - 功能入口
- **成功色**：绿色 (#22c55e) - 正常状态
- **警告色**：橙色 (#f97316) - 待处理
- **危险色**：红色 (#ef4444) - 异常/故障
- **背景色**：#f8fafc (浅灰白)
- **文字色**：#1e293b (深灰)

## 3. 页面结构

### TabBar 导航 (4个页面)
1. **首页** - 功能入口、数据概览
2. **培训资料** - 培训手册上传/查看
3. **器械巡检** - 巡检记录列表、新增记录
4. **我的** - 个人中心

### 页面层级
```
TabBar
├── 首页 (index)
├── 培训资料 (training)
│   └── 资料详情 (detail)
├── 器械巡检 (inspection)
│   ├── 新增巡检 (add)
│   └── 巡检详情 (detail)
└── 我的 (profile)
```

## 4. 组件选型原则

- 通用组件（按钮、输入框、卡片、列表）优先使用 `@/components/ui/*`
- 禁止用 `View/Text` 手搓通用组件
- 表单场景使用 Field + Input/Textarea
- 列表展示使用 Card 容器
- 状态标签使用 Badge

## 5. 功能模块

### 5.1 培训资料管理
- 资料列表展示（标题、类型、更新时间）
- 支持 PDF、Word、图片等格式
- 上传新资料
- 查看/下载资料

### 5.2 器械巡检
- 巡检记录列表（器械名称、状态、检查人、时间）
- 新增巡检记录
- 巡检状态：正常/待维修/故障

### 5.3 首页概览
- 今日待巡检数量
- 最近巡检记录
- 培训资料数量统计

## 6. 小程序约束

- 包体积限制：主包不超过 2MB
- 图片资源走 TOS 对象存储
- TabBar 图标使用本地 PNG（81x81px）
