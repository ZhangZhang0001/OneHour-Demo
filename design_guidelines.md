# 健身房内部管理系统 - 设计指南

## 1. 品牌定位

- **应用定位**：健身房内部管理工具
- **目标用户**：健身房员工、教练、管理人员
- **核心功能**：培训资料管理、器械巡检记录、匿名反馈
- **设计风格**：专业简洁、温暖亲和、专注成长

## 2. 配色方案（统一蓝灰色系）

| 用途 | 颜色名称 | Tailwind 类名 | 说明 |
|------|----------|---------------|------|
| 主色调 | 专业蓝 | `bg-slate-700` / `text-slate-700` | 标题、重要按钮 |
| 主色调 | 活力蓝 | `bg-blue-600` / `text-blue-600` | 功能入口、TabBar 选中态 |
| 次要色 | 浅蓝 | `bg-blue-50` / `bg-blue-100` | 背景装饰、hover 态 |
| 成功色 | 绿色 | `bg-green-500` / `text-green-600` | 正常状态 |
| 警告色 | 橙色 | `bg-orange-500` / `text-orange-600` | 待处理 |
| 危险色 | 红色 | `bg-red-500` / `text-red-600` | 故障、异常 |
| 背景色 | 灰白 | `bg-gray-50` | 页面背景 |
| 文字色 | 深灰 | `text-gray-700` | 正文文字 |
| 文字色 | 中灰 | `text-gray-500` | 次要说明文字 |

### 语义化颜色使用规范

```tsx
// 主色调（标题、TabBar）
<span className="text-slate-700">专业蓝</span>

// 功能入口
<span className="text-blue-600">功能蓝</span>

// 状态指示
<span className="text-green-600">正常</span>
<span className="text-orange-500">待处理</span>
<span className="text-red-600">异常</span>

// 背景装饰
<div className="bg-blue-50">浅蓝背景</div>
```

## 3. 页面结构

### TabBar 导航 (4个页面)
1. **首页** - 功能入口、数据概览
2. **培训资料** - 培训手册上传/查看
3. **器械巡检** - 巡检记录列表、新增记录
4. **我的** - 个人中心

### 页面层级
```
TabBar
├── 首页 (index)
├── 培训资料 (training)
│   └── 资料详情 (detail)
├── 器械巡检 (inspection)
│   ├── 新增巡检 (add)
│   └── 巡检详情 (detail)
└── 我的 (profile)
```

## 4. 组件选型原则

- 通用组件（按钮、输入框、卡片、列表）优先使用 `@/components/ui/*`
- 禁止用 `View/Text` 手搓通用组件
- 表单场景使用 Field + Input/Textarea
- 列表展示使用 Card 容器
- 状态标签使用 Badge

## 5. 功能模块

### 5.1 培训资料管理
- 资料列表展示（标题、类型、更新时间）
- 支持 PDF、Word、图片等格式
- 上传新资料
- 查看/下载资料

### 5.2 器械巡检
- 巡检记录列表（器械名称、状态、检查人、时间）
- 新增巡检记录（按 ABC 三区选择）
- 巡检状态：正常/有磨损/故障
- 每日自动重置

### 5.3 首页概览
- 今日待巡检数量
- 未巡检器械滚动展示
- 培训资料数量统计

### 5.4 匿名反馈
- 匿名提交建议/需求/问题
- 保护员工隐私

## 6. 小程序约束

- 包体积限制：主包不超过 2MB
- 图片资源走 TOS 对象存储
- TabBar 图标使用本地 PNG（81x81px）
